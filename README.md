<h2> Summary</h2>
This is a brief walk through on how to connect a Microsoft Azure Virtual Machine (VM) to a Virtual Private Network (VPN) using Proton VPN. This will be based on the main computer using a macOS. <br />

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machine)
- Microsoft Remote Desktop
- Proton VPN

<h2>Operating System(s) Used </h2>

- Windows 10 Pro</b> 

<h2>List of Prerequisites</h2>

- A Subscription with Microsoft Azure (FREE $200 USD worth of credit!)
- Microsoft Remote Desktop (FREE!)
- An Account with Proton VPN (FREE!)
  
<h2>Installation Steps</h2>

<p>
<img src="https://i.imgur.com/fCdPrLW.png" height="80%" width="80%" alt="Create VM"/>
</p>
<p>
In Microsoft Azure, create a Virtual Machine. Name the Resource Group and Virtual Machine. You can set the region to a location other than where you live. For this instance, the region has been set for North Europe. Select Windows 10 Pro for Operating System. Select an appropriate size Virtual Machine, such as Standard with 2vcpus and GiB memory. Create a User Name and Password. (Be sure to check the license agreement box in the bottom left hand corner. Then create. The above image is a VM generated by Microsoft Azure. 
</p>
